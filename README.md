# Desafio Streaming Churn - Probabilidade e Amostragem

![Python](https://img.shields.io/badge/Python-3.x-blue)
![Pandas](https://img.shields.io/badge/Pandas-Data%20Analysis-green)

Terceiro desafio em squad do Bootcamp Data Analytics 2026.1, oferecido pela [WoMakersCode](https://womakerscode.org). Desenvolvido para aprofundar o conhecimento em Probabilidade e Amostragem de Estatística com Python, analisando perfil de cancelamento de assinatura em empresa de streaming.

> **🎯 Objetivo:** Analisar a base de dados da empresa de streaming para saber se o perfil dos clientes que cancelam varia conforme o estágio da jornada.

## ⚙️ Como rodar o projeto

1. Clone o repositório

2. Instale as dependências:

  ``` bash
  pip install -r _dependencias.txt

  # OU

  pip install pandas jupyter ipykernel
  ```

3. Execute o Notebook:

- Abra `analise.ipynb` no VS Code
- Selecione o kernel no canto superior direito
- Execute tudo ou célula por célula

## 🚀 Tecnologias Utilizadas

- Python 3.x
- Pandas — manipulação e análise de dados
- Jupyter Notebook / Google Colab — ambiente interativo
- VS Code + extensões Python e Jupyter
